\paragraph{1. Scatter Plot \& Outlier Identification}

A scatter plot of $y$ (number of insured persons) versus $x$ (years since 1987) reveals
a generally decreasing trend from $\approx 12{,}400$ down to $\approx 7{,}200$.
However, one data point stands out sharply:
\begin{itemize}
  \item $x = 3$ (year 1990): $y = 1050$
\end{itemize}

This value is dramatically lower than the surrounding data, which are all within
approximately $7{,}200$ to $12{,}400$. The sharp drop breaks the otherwise smooth
downward trend. This observation is therefore identified as a clear outlier, most
likely due to a data entry error (e.g.\ $10{,}500$ instead of $1{,}050$).

\paragraph{2. Regression Models (WITH Outlier)}

We apply both OLS and matrix-form methods to compute linear model parameters,
then use the matrix form for quadratic and cubic models.

\subparagraph{2.1 Linear Model}

\textbf{Hand Analysis (OLS Method)}

The equation for the linear model is:
\begin{equation}
  \hat{y}_i = \beta_0 + \beta_1 x_i
\end{equation}

The key formulas:
\begin{equation}
  \hat{\beta}_1 = \frac{\displaystyle\sum_{i=1}^{n} x_i y_i - \frac{\bigl(\sum x_i\bigr)\bigl(\sum y_i\bigr)}{n}}
                       {\displaystyle\sum_{i=1}^{n} x_i^2 - \frac{\bigl(\sum x_i\bigr)^2}{n}},
  \qquad
  \hat{\beta}_0 = \bar{y} - \hat{\beta}_1\,\bar{x}
\end{equation}

$n = 10$,\quad $\bar{y} = 83350/10 = 8335$,\quad $\bar{x} = 45/10 = 4.5$

\[
\hat{\beta}_1 = \frac{344{,}750 - \dfrac{45 \times 83{,}350}{10}}{285 - \dfrac{45^2}{10}} = -367.57
\]

\[
\hat{\beta}_0 = 8335 + 367.57 \times 4.5 = 9989.09
\]

\textbf{Hand Analysis (Matrix Form)}

\[
\bm{\beta} = (X^T X)^{-1} X^T Y
\]

\[
\bm{\beta} = \begin{pmatrix}\beta_0\\\beta_1\end{pmatrix},\quad
X = \begin{pmatrix}1&0\\1&1\\1&2\\1&3\\1&4\\1&5\\1&6\\1&7\\1&8\\1&9\end{pmatrix},\quad
Y = \begin{pmatrix}12400\\10900\\10000\\1050\\9500\\8900\\8000\\7800\\7600\\7200\end{pmatrix}
\]

\[
\begin{pmatrix}\beta_0\\\beta_1\end{pmatrix} = \begin{pmatrix}9989.09\\-367.576\end{pmatrix}
\]

Linear equation:
\begin{equation}
  \hat{y}_i = 9989.09 - 367.576\,x
\end{equation}

\textbf{R² Verification Table}

\[
\bar{y} = \frac{\sum y_i}{n} = \frac{83350}{10} = 8335
\]
